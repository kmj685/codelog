## 학습 내용

① 핵심 개념: 변수는 데이터를 저장하는 메모리 공간으로, 데이터 타입은 변수에 저장된 데이터의 형태를 정의합니다.
② 작동 원리: 변수에 값을 할당하면, 컴퓨터는 해당 값을 메모리에 저장하고, 변수 이름을 사용하여 값을 참조할 수 있습니다.
③ 실전 적용: 변수와 데이터 타입은 모든 프로그래밍 언어에서 사용되며, 예를 들어 Flutter에서 텍스트 필드의 입력 값을 저장하고 처리할 때 변수를 사용합니다.
④ 주의사항: 변수의 데이터 타입을 잘못 선언하면, 런타임 오류가 발생할 수 있습니다.
⑤ 핵심 요약: 변수와 데이터 타입은 프로그래밍의 기초이며, 변수를 선언하고 데이터 타입을 이해하는 것이 중요합니다.

## 코드 예시

```
```dart
// 변수 선언과 초기화
int num = 10; // 정수형 변수
String name = 'John'; // 문자열 변수
print(num); // 10
print(name); // John
```
